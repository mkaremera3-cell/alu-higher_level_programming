# Python - Input/Output

This project covers file handling and JSON serialization/deserialization in Python.

## Files

| File | Description |
|------|-------------|
| `0-read_file.py` | Reads a UTF-8 text file and prints to stdout |
| `1-write_file.py` | Writes a string to a file, returns number of chars written |
| `2-append_write.py` | Appends a string to a file, returns number of chars added |
| `3-to_json_string.py` | Returns JSON string representation of an object |
| `4-from_json_string.py` | Returns Python object from a JSON string |
| `5-save_to_json_file.py` | Saves a Python object to a file as JSON |
| `6-load_from_json_file.py` | Loads a Python object from a JSON file |
| `7-add_item.py` | Adds command-line arguments to a JSON list file |
| `8-class_to_json.py` | Returns dictionary description of an object for JSON serialization |
| `9-student.py` | Student class with to_json() method |
| `10-student.py` | Student class with filtered to_json(attrs) method |
| `11-student.py` | Student class with reload_from_json() for deserialization |
| `12-pascal_triangle.py` | Returns Pascal's triangle as list of lists |

## Learning Objectives
- How to open, read, write, and append to files in Python using with
- What JSON is and how to serialize/deserialize Python objects
- How to convert a class instance to a JSON-compatible dictionary
- How to implement Pascal's Triangle
