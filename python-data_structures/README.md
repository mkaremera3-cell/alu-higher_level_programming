# Python - Data Structures: Lists, Tuples

This project covers Python data structures including lists and tuples.

## Functions

- **0-print_list_integer.py**: Prints all integers of a list
- **1-element_at.py**: Retrieves an element from a list
- **2-replace_in_list.py**: Replaces an element in a list
- **3-print_reversed_list_integer.py**: Prints integers of a list in reverse
- **4-new_in_list.py**: Replaces element without modifying original list
- **5-no_c.py**: Removes all c and C from a string
- **6-print_matrix_integer.py**: Prints a matrix of integers
- **7-add_tuple.py**: Adds 2 tuples
- **8-multiple_returns.py**: Returns tuple with length and first character
- **9-max_integer.py**: Finds the biggest integer in a list
- **10-divisible_by_2.py**: Finds multiples of 2 in a list
- **11-delete_at.py**: Deletes item at a specific position
- **12-switch.py**: Switches the values of two variables
