# Python - More Classes and Objects

This project covers more advanced Object-Oriented Programming (OOP) concepts in Python, focusing on the Rectangle class and its progressive development.

## Tasks

- **0-rectangle.py**: Empty Rectangle class
- **1-rectangle.py**: Rectangle with width and height attributes
- **2-rectangle.py**: Rectangle with area and perimeter methods
- **3-rectangle.py**: Rectangle with string representation using `#`
- **4-rectangle.py**: Rectangle with `__repr__` for eval() recreation
- **5-rectangle.py**: Rectangle with deletion message
- **6-rectangle.py**: Rectangle with instance counter
- **7-rectangle.py**: Rectangle with customizable print symbol
- **8-rectangle.py**: Rectangle with `bigger_or_equal` static method
- **9-rectangle.py**: Rectangle with `square` class method
